## Build with

## What I learned

- DOM manipulation
- Objects manipulation (creating/reading/updating/deleting objects)
- Clamp function
- Light/dark mode

## Project features

- Listing all books
- Adding new books
- Editing/removing existing books
- Toggle light/dark mode switch
- SVG color manipulation
